=== Рекурсивне обчислення НСД масиву ===
Дано натуральні числа m, n₁, n₂, ..., nₘ (m>1).
Обчислити НСД(n₁, n₂, ..., nₘ) рекурсивно.
Властивість: НСД(n₁, n₂, ..., nₖ) = НСД(НСД(n₁, n₂, ..., nₖ₋₁), nₖ)
Приклад: НСД(12, 18, 24) = НСД(НСД(12, 18), 24) = НСД(6, 24) = 6

Введення чисел

Діапазон: від 2 до 50
